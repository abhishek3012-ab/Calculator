# Calculator

A simple calculator web application built using **HTML, CSS, and JavaScript**.

## Features
- Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
- Responsive design for mobile and desktop
- Clean and simple UI
- Keyboard support for quick calculations

## Demo
Check out the live demo: https://abhishek3012-ab.github.io/Calculator/

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/calculator.git
   ```
2. Navigate to the project folder:
   ```sh
   cd calculator
   ```
3. Open `index.html` in your browser.

## Usage
1. Enter numbers using the calculator buttons or your keyboard.
2. Use the `+`, `-`, `*`, and `/` operators for calculations.
3. Press `=` to see the result.
4. Click `C` to clear the input.


## Technologies Used
- **HTML** – Structure of the calculator
- **CSS** – Styling and layout
- **JavaScript** – Functionality and logic
